# **사용자 정의 타입**
- **사용자 정의 타입**(custom type)은 Go의 기본 타입 외에 **추가로 타입을 정의**할 때 사용한다.
- 또한 **type**이라는 키워드를 사용하여 **생성**한다.
    ~~~go
    type 타입명 타입명세
    ~~~
- **타입명**은 **패키지**나 **함수** 내에서 **유일**해야 한다.

- **타입명세로**는 **다음 목록**들을 사용할 수 있다.
    - **Go의 기본 타입**(string, int, slice, map, channel 등)
    - **구조체**
    - **인터페이스**
    - **함수 서명**
